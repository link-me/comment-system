Назначение: небольшой сервис комментариев для демонстрации портфолио.
Решаемые задачи: хранение и управление комментариями, фильтрация по постам/авторам, простая модерация (флаг).

Как запускать:
1) Установить Node.js (версия 18+)
2) В папке проекта выполнить: `npm install`
3) Запустить сервер: `npm start`
4) Проверить: открыть `http://localhost:3000/health` или использовать примеры из README

Примечания:
- Данные сохраняются в `data/comments.json`
- Для демонстрации не требуется база данных
- При переносе в прод: добавить пагинацию, лимиты, аутентификацию
