# ROADMAP (RU)

## Цели
- Базовый CRUD по комментариям
- Флаг модерации
- Простая файловая база без внешних сервисов

## Выполнено
- Express-сервер
- CRUD эндпоинты
- Валидация входных данных
- Логирование запросов (morgan)
- Документация и примеры

## В процессе / План
- Пагинация в `GET /comments`
- Поле `parentId` для тредов
- Экспорт/импорт данных
- Хранилище: переход на SQLite/PG при необходимости
- Dockerfile и compose

## Вехи
- v0.1 — MVP API с файловым хранилищем
- v0.2 — Пагинация и треды
- v0.3 — Перенос хранилища на SQL
